## Built With

This website is built with ❤️ on ReactJS and AWS S3, statically typed in TypeScript

# Installation

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).
Install dependencies

```
$ npm install
```

Build Server

```
$ npm build
```

Run Server

```
$ npm start
```
